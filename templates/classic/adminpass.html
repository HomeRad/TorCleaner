Administrator password for WebCleaner Proxy

The administrator password was not yet initialized. This password protects the web configuration pages. Without an administrator password, every user can change the proxy configuration.

Please open the file /etc/webcleaner/webcleaner.conf and replace the line adminpass="" with: 

adminpass="dummy"

After that, restart WebCleaner and click on the configuration link below. Your new credentials for the configuration are:
username user
password pass

The configuration also allows you to change the password.
